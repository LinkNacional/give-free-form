Form styling plugin for GiveWP.

== Description ==

The (Donation Form Customization for GiveWP)[https://www.linknacional.com.br/wordpress/givewp/give-free-form/] is an extension plugin that's give an option to edit the CSS values of donation form, inside of GiveWP forms configuration.

The form customization is not relevant for Multi-step form of GiveWP. If you want to use the Donation Form Customization for GiveWP, you need to change the Form Template to the "Legacy" option.


**Dependencies**

Donation Form Customization for GiveWP plugin is dependent on [Give plugin](https://givewp.com/), please make sure Give is installed and properly configured before starting Donation Form Customization for GiveWP installation.


**User instructions**

1. In the Wordpress sidebar, look for the "Donations" option and select it;

2. In the page "Donation Forms", look for the donation form - preferably of the legacy template - to customize and click on "Edit";

3. In the Donation Form Option sidebar, click on "Form Customization" option;

4. Select the first field "Enable" as Enabled;

5. Configure the rest of the form customization as you needs;

6. Then, click on the "Publish" (If it is already published, it will be the "Update" button);

The Donation Form Customization for GiveWP is now live and working.


== Installation ==

1. In the Wordpress sidebar, look for the "Plugins" option and select it;

2. Press the button "Add new" next to the "Plugins" title at the top of page;

3. Click on the "Submit Plugin" option in the page title and upload the lkn-give-free-form.zip plugin;

4. Click on the "Install now" button and then activate the installed plugin;

At finish this steps, the Donation Form Customization for GiveWP is now activated and ready to be used.
