Form styling plugin for GiveWP

== Description ==

This plugin give a option to edit the CSS values of donate form, inside of configuration GiveWP forms.

The custom form is not relevant to the Multi-Step form of giveWP. 
If you want to use the Free Form Plugin it is necessary to change the Form Template to "Legacy" option.

== Frequently Asked Questions ==

= HOW TO USE =

Install the plugin, activated the plugin. Go to forms, edit a form, inside the opções go to "Estilização do Formulário".
